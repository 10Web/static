中间件使用(middleware)

在honeybee应用框架中，使用读取配置的方法加载中间件，因此绝大多数的中间件只需修改配置文件即可使用，例如加载webpack中间件：

  // config/config_default.js
  // ...
  middleware: {
    webpack: {
      enable: true,
      module: 'honeypack',  // 需要通过tnpm i honeypack --save来安装依赖
      router: '/assets'
    },
    spa: {
      enable: true
    }
  }
  // ...
